# ShadowFiles

ShadowFiles: Encrypted Crime Report System is a C++-based security project that securely stores and manages criminal investigation records using file handling and Vigenère Cipher.

## 🔐 Features
- Data Encryption using Vigenère Cipher  
- Secure File Storage  
- Password Authentication (3 attempts)  
- Password Strength Check  
- Add, View, and Search Records  

## ⚙️ Technologies Used
- C++  
- File Handling  
- Encryption (Vigenère Cipher)  

## ▶️ How to Run
1. Open the .cpp file in Dev C++ / VS Code  
2. Compile the program  
3. Run the executable  
4. Enter password and use menu  

## 📁 Project File
- `Shadowfiles IS project.cpp` → Main source code  

## 💡 Concept
This project demonstrates how sensitive crime data can be securely stored and accessed using encryption techniques.
